Attached are all the third-parties useful to develop on wogedit. This is provided for convenience, you can easily download them (and compile them) yourself.

All binary are 32bits Windows executable for python 2.5 (pyqt is not yet provided as a stable for python 2.6).

This includes (listed in installation order):
* python-2.5.2.msi: from http://www.python.org
* pywin32-212.win32-py2.5.exe, from http://sourceforge.net/project/showfiles.php?group_id=78018&package_id=79063
* pycrypto-2.0.1.win32-py2.5.exe: precompiled based on source from http://www.amk.ca/python/code/crypto.html, generated using python setup.py bdist_wininst.
* PyQt-Py2.5-gpl-4.4.3-1.exe, from http://www.riverbankcomputing.co.uk/software/pyqt/download
* py2exe-0.6.8.win32-py2.5.exe, from http://sourceforge.net/projects/py2exe/
